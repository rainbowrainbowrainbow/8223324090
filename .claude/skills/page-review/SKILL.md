# Page UX Review — Парк Закревського Періоду

Проведи повне UX-рев'ю сторінки $ARGUMENTS.

## Сторінки проекту
- `index.html` — Головна SPA (таймлайн, бронювання, модалки, налаштування)
- `kleshnya.html` — Чат з Клешнею (AI-асистент)
- `tasks.html` — Дошка задач
- `programs.html` — Каталог програм
- `staff.html` — Графік аніматорів
- `designs.html` — Борд дизайнів (drag & drop зображення)
- `invite.html` — Публічне запрошення

## Чек-лист перевірки

### 1. Структура HTML
- [ ] Семантичні теги (header, main, nav, section, article)
- [ ] Правильна ієрархія заголовків (h1 → h2 → h3)
- [ ] Немає порожніх div без семантики
- [ ] Alt-тексти на всіх зображеннях
- [ ] Lang="uk" на html тезі

### 2. Доступність (a11y)
- [ ] role атрибути на інтерактивних елементах
- [ ] tabindex для навігації клавіатурою
- [ ] aria-label на іконкових кнопках
- [ ] Контраст тексту достатній (WCAG AA: 4.5:1)
- [ ] Focus-стилі видимі

### 3. Мобільна версія
- [ ] Viewport meta tag правильний
- [ ] Немає горизонтального скролу
- [ ] Touch targets мінімум 44px
- [ ] Шрифт інпутів мінімум 16px
- [ ] overscroll-behavior на scroll-контейнерах
- [ ] Адаптивна навігація

### 4. Продуктивність
- [ ] CSS/JS з cache-bust параметрами (?v=X.X)
- [ ] Зображення оптимізовані
- [ ] Немає блокуючих скриптів в head
- [ ] Lazy loading де можливо
- [ ] Мінімум inline стилів (краще в css/)

### 5. Dark mode
- [ ] Early dark mode script (prevent flash)
- [ ] initDarkMode() виклик після config.js
- [ ] Всі елементи мають dark-mode стилі
- [ ] Тіні адаптовані для темного фону
- [ ] Зображення не засліплюють в dark mode

### 6. Безпека
- [ ] JWT токен з localStorage, не в URL
- [ ] XSS-захист (textContent замість innerHTML де можливо)
- [ ] CSRF-захист на POST запити
- [ ] Санітизація user input
- [ ] Немає hardcoded credentials

### 7. UX патерни
- [ ] Loading стани (skeleton/spinner)
- [ ] Error стани (user-friendly повідомлення)
- [ ] Empty стани (коли немає даних)
- [ ] Success feedback (після дії)
- [ ] Підтвердження деструктивних дій

### 8. Консистентність з проектом
- [ ] Навігація однакова на всіх сторінках
- [ ] Стиль кнопок відповідає дизайн-системі
- [ ] Формати дат/валюти правильні
- [ ] Мова UI українська
- [ ] Cache-bust версія актуальна
